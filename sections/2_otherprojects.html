Other Projects

Masterthesis

Crowd evacuation considering psychological and physical behavior using realtime physics built for videogames.

In my masterthesis I examined the psychological behavior and the physical impact of a crowd's member on the crowd. I developed a real-time simulation for a crowd of poeple using Unity3D and C#. The developed system contains a behavior-AI for the agents with a psychological and physical state. The physical interaction was done by using Unitys physic engine and the movements of the agents was realised with a navigation mesh and an algorithm for collision avoidance.

The developed simulation is able to show physical impacts such as crowding and crushing of agents, interaction of agents with their environment as well as factors of stress.

You can see the simulation in action by following this link.

Sage-sb

Semiautomatic generation of semantic building models from image series.

Sage-sb is an "... approach to generate a 3D model of a building including semantic annotations from image series. ... This approach has been implemented as a plugin for the photostitching tool Hugin. ... The resulting model contains semantic, geometric and appearance information and is represented in City Geography Markup Language (CityGML)." - project description on sourceforge.net

Sage-sb is written in C++ using wxWidgets as the GUI part with an embedded 3D-View which was done in OpenGL.

I worked on this project during my work at AGAS. My main area was building the GUI and I was involved in planning the project. For more information have a look at the project and code at sourceforge.net.

Robot Face

Robot Face is a talking head that can talk with synchronized mouthmovements and display its spoken text. It is also able to show emotions. In further development it was also able to show an image or a videostream instead of its face.

I developed it for the robot Lisa of AGAS at the University of Koblenz during my Bachelor thesis. We released a package for the robot operating system (ROS). More details are here.

It was written in C++ with Ogre3D embedded in a Qt application. The face itself was modeled in Blender. Festival was used for the text-to-speech part.

If you are intersted in robot Lisa you can find more information on the official website.
